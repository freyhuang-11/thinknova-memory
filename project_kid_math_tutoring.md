用户（非专业老师）每周给一个孩子上 **3 次** 数学网课，视频会议共享屏幕。我负责**持续产出课件**：每次课一个**自包含 HTML 文件**（点击「显示思路」才展开答案，避免提前剧透），放在桌面 `C:\Users\samso\Desktop\`。

**孩子情况**：上海金山外国语小学，2026.06 刚考完二年级期末，9 月升三年级。数学**计算没问题，但逻辑/思路弱**——错题集中在：边界「差一」(植树/班次/「比…多比…少」)、余数最大(余数比除数小1)、多步骤题第一步错带垮全题、逆推还原、分类讨论(漏情况)、审题(整百/整十/相邻数概念混)。

**⚠️ 重心已更正（第1课上完后用户实测反馈）**：真正的短板是 **① 基础计算准确度（算错、粗心，如 780−85 算成 605）② 读题/审题不清**。**奥数他全对、是强项**，逻辑并不弱——最初"主攻逻辑"是用户的猜测，已被推翻。
**新重心 = 计算准确度 + 审题。** 奥数降为"调味/奖励"，每课放一两道保持兴趣即可。三年级上预习继续穿插。暑假窗口：练扎实计算 + 读题 + 预习三年级。

**课件格式**（已定型，见首课 `二年级数学网课_资料.html`）：时间表 + 「🗣️四句万能提问」家长话术框 + 每题卡片(题目→孩子做→点击四步思路①圈关键②画图列式③想为什么④验算→答案)，原错题标粉色「考试错」。教学法核心：**用"问"代替"讲"**，让孩子说出思路。

**奥数题必须用"没考过的新题"**（用户反馈：第一课奥数照搬了考卷，他觉得"都考过了、没新鲜感"）。规矩：考卷原错题只放在**思路板块的「订正区」**（标"考试错"，本就该重做）；**奥数区一律换新场景新数字的新题**，但保持同一方法/套路（如植树问题换成挂灯笼/排队报数）。
