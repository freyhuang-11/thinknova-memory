**桌面(`D:\SamsoData\Desktop`)只放要给老板当场看的测试成片(mp4)/成图(海报)。** 其他一切文档(给技术交付/诊断/清单/报告)一律进 `D:\SamsoData\Documents\视频制作平台分析\` 对应子目录:
- 给技术/交付类 → `02_交付内容\`(所有 `给技术_…md` 都在这)
- 诊断/待办 → `01_问题诊断\`
- 规格与参考 → `00_规格与参考\`

**Why**:2026-07-12 我把「给技术_实体店视频3问题+多图需求」MD 直接丢桌面,老板问"你为什么乱放"。桌面是给老板看成片/成图的地方,文档丢桌面=乱。

**How to apply**:生成文档前先想清归属子目录,别默认桌面;只有"要老板肉眼验收的成片/成图"才进桌面。参见 [[reference-thinknova-paths]] 环境坑。
