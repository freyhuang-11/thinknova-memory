# 现实校准（必读）

- **TikTok 私信不是 1 天 1 万条**。是按周 + GMV 分层：起步包 1000 一次性 / $0-2k = 2000/周 / $2k-50k = 7000/周 / >$50k = unlimited。**只算 first-contact**，老对话回复不算
- 新客户第一周 ≤ 1000 条 ≈ 140/天
- 服务商累加：50 客户 × 各自店铺 quota，按 shop_cipher 隔离
