ThinkNova 实体店 海报/短视频 agent 是**内容制作工具**，目标 = **客户生成的内容他们自己满意 + 能吸引顾客**，**不替商家做合规审查**。

**Why:** 用户 2026-07 明确拍板。之前我给新行业(中医理疗/娱乐)自作主张加了合规禁令(中医禁疗效/病名、娱乐禁低俗/色情/酗酒)，被用户否掉："中医就得加疗效，娱乐就得加低俗，目的只有吸引客户"。写提示词/负面时别把"广告法合规"当默认约束加进去。

**How to apply:**
- 中医理疗 → 突出理疗效果/体感改善/养生招牌，**不删疗效**。
- 娱乐(KTV/酒吧) → 氛围拉满/潮流性感/夜生活魅力，允许低俗擦边。
- 其余行业同理：吸引优先，别加"禁夸大/禁疗效/禁低俗"这类合规负面。
- **不要一刀切 ban 未成年/儿童**：用户点破"万一人家想要小朋友玩得开心呢"——家庭KTV/儿童游乐/亲子店/生日派对都是正经内容，blanket"未成年/儿童"负面会误杀。已从娱乐负面里去掉。我唯一不做的是"性化未成年"(CSAM，无需写进配置、正常营销也不会触发)，别再拿"未成年"当泛化约束。
- 平台审核(抖音/小红书/美团)对医疗疗效、色情会限流封号——这个只作"善意提醒"给用户，不作为我主动加的约束。
- **第一版打海外**：平台优化顺序=海外优先(通用→TikTok→Instagram→Facebook→YouTube Shorts→Pinterest→X→抖音/小红书/视频号/快手)。已删/隐藏 B站+微博、微信标签改"视频号"。
- **平台编辑机制(实测)**：platform 选项有 `enabled` 字段。**海报删数组会被服务端塞回→必须 enabled:false 隐藏**；视频删数组能生效。sortOrder 控制前端显示序(能存)。
- **默认出图语言改不动**：copyLanguage 无 default 字段、数组序被服务端强制 zh_cn 打头，配置改不了默认语言。要英文默认=技术，或前端按 UI locale 自动匹配(海外 UI=en 可能已自动)。
- **图上文字=矢量/叠加层方向**：AI 模型只出栅格图、文字烧进像素(所以中文会糊)。真正解=AI 出视觉+预留文字区，另起 SVG/HTML 层渲染店名/价格/CTA(矢量、正确、可编辑、任意语言)。=技术工程改造，配置只能配合"出干净留白背景"。一举解决糊字+多语言上图。
